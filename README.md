# sway layout indicator
> swaylay

swaylay is a layout indication utility for
[sway](https://github.com/swaywm/sway).

## Building

Install dependencies:

* git \*
* meson \*
* sway

_\*Compile-time dep_

Run these commands:

```shell
$ meson build
$ ninja -C build
$ sudo ninja -C build install
```
